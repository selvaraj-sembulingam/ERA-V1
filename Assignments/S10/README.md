# Residual CNN Model

This folder contains an implementation of a Custom ResNet model

## Folder Structure
```
└── README.md
└── src/
    └── data_setup.py
    └── utils.py
    └── engine.py
    └── custom_resnet.py
└── models/
    └── CustomResNet.pth
    └── incorrect_images.png
    └── loss_accuracy_plot.png
    └── lr_finder_plot.png
└── train.py
└── S9.ipynb
```

## How to Run the code
Clone the repo and run
Change your current directory to S9
```
python train.py
```
